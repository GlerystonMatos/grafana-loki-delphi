# grafana-loki-delphi
Exemplo de integração do Grafana, Grafana Loki e uma aplicação Delphi.

## Endereços disponíveis

Pagina com informações do Grafana Loki:

```bash
http://localhost:80/loki/api/v1/status/buildinfo
```

Pagina inicial do Grafana:

```bash
http://localhost:3000/
```
